# MiniWorld存档目录结构

File 描述
roles 角色背包
objlibs 物体库
mods 模组
m0 主世界数据
m1 次世界数据
m2 宇宙数据
m3 预留
custommodel 自定义模型
blueprint 蓝图
cspermit 通行证
timers 计时器
transfer 传送门
ss 逻辑
Living 测试
thumb.png_ 存档封面
__thumb.webp_ 存档webp封面
triggerarea.fb 触发区域
triggerposition.fb 触发位置
wdesc.fb 存档描述
wglobal.fb 世界规则
wmultilang.fb 多语言
wsize.fb 存档世界边境
wterrtype.fb 存档地形
vehicle 载具
{path}/mods/{name}{uuid}/logical/ 游戏逻辑包
blob_storage blob储存
data\micro\loadmicrores.xml 校验资源md5
shadercompilecase.ini 着色器编译实例(模板)
data 实时数据
locales 本土化语言包
log 日志文件
swiftshader SwiftShader用于CPU 上进行高性能图形渲染的软件库
maindata 分布式哈希算法分割打散的游戏主要数据
update 更新临时目录
\pluginres\ 插件
\pluginres\skin 皮肤
