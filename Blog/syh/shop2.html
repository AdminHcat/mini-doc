    # 【商业化】开发者如何设置商城游戏道具-解密跑酷篇

    这篇文章主要介绍如何在跑酷地图和解密类地图里面设置付费点，值得注意的是和我在上一篇文章强调的一样，商城是帮助玩家克服游戏中的困难，获得更好的游戏体验的。

    试想一下，假如玩家在某个解密的关卡被卡住，无法通关，或者在某个跑酷的环节跳不过去，那么这一局游戏玩家的体验是不是很糟糕？纵然你的玩法很好，地图很漂亮，但是玩家无法通关，他就完全体验不到这种快乐。

    所以这个时候就需要游戏地图的创作者提供商店内的辅助道具，帮助玩家顺利通关。

    # 解谜类地图的商店道具设置

    解谜类地图基本都是以一个故事为主线，结合电路，隐藏点、提示等，使玩家完成一个个解密任务，并通关。 他的商店商品设置主要在一下几个方面: (1) 必须购买通行证才可解密。这个方法比较粗暴，不太建议使用，就是玩家玩之前必须付费才能游玩。

    (2) 解密辅助，如果玩家被关卡卡住，可以购买解密辅助道具(比如梯子和开关)，或者相应的解密说明。

    单纯的列出可能比较难以理解，这里结合千咫韩信的墓地探险地图说明。

    # 玩法设置方面：

    出售通关攻略，帮助卡关玩家通关 玩家进入游戏后以弹窗的方式提醒玩家购物，而不是浮现文字。

    玩家被击杀后弹窗提示购买商品 玩家出生时，默认提供一把攻击力25的短剑，1个夜视孢子，1个星星礼包，2个香溢麦包(不要提供太多的东西给玩家)

    # 跑酷类地图商店设置

    跑酷类地图以操作为主，完成较高难度的地图。有些情况下因为地图创造者设置不合理，会导致部分玩家无法通过或者手残玩家跳不过去。

    付费点为: (1)效果增强，道具赋予玩家更高的跳跃能力和速度 (2)辅助道具，方块，梯子和开关帮助玩家完成简单的攀爬等操作(按个购买，看视频获得) (3)工具，喷射抓钩，火箭背包，冲天炮，降落伞(将飞行生物改造，借助微缩模型改造为降落伞)

    这里结合粉红豹的风景跑酷，实例讲一下，如何设置跑酷的商品道具。他的地图商品设置是这个样子的。喷射抓钩和跳一跳果汁帮助玩家通过难度较高的关卡，宝箱钥匙配合地图上随机出现的宝箱，让玩家有更强的购买欲望。

    名称 喷射钩爪 生命药水 抗火药水 宝箱钥匙 跳一跳果汁
