# 关于方块Data

方块Data包括：方块朝向、方块色彩、激活状态，结构上类似于MC的NBT标签

# 木床

横放：床头6 床位2

反向横放：床头7 床尾3

竖放：头4尾0

反：头5尾1

# 开关

开10 关2

# 蛋糕、饲料等有食用程度的方块

Data 1-5为蛋糕被食用程度，设为6后会在原地留下无碰撞箱的空气（可用地形编辑器替换掉）

# 地心传送方块

1为正常方块态，0为隐形激活状态

# 树苗

0未生长 2生长
